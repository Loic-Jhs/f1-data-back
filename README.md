# f1-data-back

## Installation

```bash
python -m venv venv
source venv/bin/activate
# For windows
.\venv\Scripts\Activate.ps1
```

```bash
pip install -r requirements.txt
```

## Usage

```bash
uvicorn main:app --reload
```
